# Monkey

[Infinite monkey theorem](https://en.wikipedia.org/wiki/Infinite_monkey_theorem)

It's a monkey that types enough random characters as to maybe contain _The Tragedy of Hamlet_ by Shakespeare. 

If `monkey_shows.py` is executed (use `python3`) you will be able to command the monkey to type. The monkey will write its art in `isithamlet.txt` and show you what it has written in 15 line pages. **If you want to keep the monkey's piece of art you should copy or move the file `isithamlet.txt` from the folder with the source code into another folder**. The code will always overwrite the file `isithamlet.txt`.

You can also compile and execute `monkey.c` so that the monkey only writes in the file and doesn't show you anything.
