# Monkey

It's a monkey that types enough random characters as to maybe contain _The Tragedy of Hamlet_ by Shakespeare. 

If `monkey_shows.py` is executed you will be able to command the monkey to type. The monkey will write its art in `isithamlet.txt` and show you what it has written in 15 line pages. If you want to keep the monkey's piece of art copy or move the file `isithamlet.txt` from the folder with the source code, the code will always replace the file `isithamlet.txt`.

You can also compile and execute `monkey.c` so that the monkey only writes in the file and doesn't show you anything.
